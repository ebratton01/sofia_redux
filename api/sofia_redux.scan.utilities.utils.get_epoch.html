get_epoch¶

sofia_redux.scan.utilities.utils.get_epoch(equinox)[source]¶

Return an astropy frame representing the equinox of an observation.

An FK4 frame will be returned in instances where equinox is a string and the first letter is ‘B’ (B1950 for example), or the year is less than 1984. In other instances a Julian FK5 frame will be returned.

Parameters:
equinoxBaseCoordinateFrame or Time or float or int or str

The equinox of the observation

Returns:
epochBaseCoordinateFrame

Typically and FK5 or FK5 frame unless another BaseCoordinateFrame was passed in as equinox.
