get_channel¶

sofia_redux.instruments.fifi_ls.split_grating_and_chop.get_channel(hdul, channel_index=3)[source]¶

Returns RED or BLUE channel extracted from header and data.

If the channel cannot be read from the prime header, it is read from the data header.

Parameters:
hdulfits.HDUList
channel_indexint, optional

data header location determining channel

Returns:
str

RED, BLUE, or UNKNOWN
