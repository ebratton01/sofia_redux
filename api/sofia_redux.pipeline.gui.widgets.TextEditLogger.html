TextEditLogger¶

class sofia_redux.pipeline.gui.widgets.TextEditLogger[source]¶

Log handler for a TextEdit-based logging window.

Uses CustomSignals to emit a ‘finished’ signal to a display slot.

Initialize the logger.

Methods Documentation

emit(record)[source]¶

Emit a log message.

Messages are formatted to HTML with font colors for the level string (green for DEBUG, orange for WARNING, red for ERROR). A finished signal (from CustomSignals) is emitted, containing the formatted message.

Parameters:
recordlogging.LogRecord

The log record, with an additional ‘origin’ attribute attached by astropy.log.
