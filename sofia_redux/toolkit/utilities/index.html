Utilities (sofia_redux.toolkit.utilities)¶

Introduction¶

The sofia_redux.toolkit.utilities module contains a few miscellaneous functions that are used in other SOFIA pipelines or functions in sofia_redux.toolkit.

Miscellaneous Functions (func)¶

The sofia_redux.toolkit.utilities.func module contains various functions that are used by multiple other functions in sofia_redux.toolkit or by the SOFIA pipeline, but are not easily classified into one of the larger modules.
