interp_1d_point¶

sofia_redux.toolkit.interpolate.interpolate.interp_1d_point(x, y, xout)[source]¶

Perform linear interpolation at a single point. x must be monotonically increasing or decreasing containing unique values.

Superfast compared to anything else including numpy.interp.

Parameters:
xarray_like of float

(N,) array of independent values. Must be monotonically increasing or decreasing.

yarray_like of float

(N,) array of dependent values.

xoutfloat

New independent value.

Returns:
youtfloat

The new dependent values at xout.
