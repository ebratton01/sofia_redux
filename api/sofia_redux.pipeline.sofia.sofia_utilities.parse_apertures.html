parse_apertures¶

sofia_redux.pipeline.sofia.sofia_utilities.parse_apertures(input_position, nfiles, error_message=None, allow_empty=False)[source]¶

Parameters:
input_positionstr

Input parameter string.

nfilesint

Number of input files expected.

error_messagelist of str, optional

Error message to display if parsing fails.

allow_emptybool

If set, an empty list is allowed for some files.

List of length nfiles, containing lists of floating point aperture values.
